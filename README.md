# Todo List

## Description

This is a simple Todo List application built with React using Vite. The app shows a list of todo items and is part of a React learning assignment.

## Installation

1. Clone the repository.
2. Open the project folder in your terminal.
3. Install the project dependencies by running:

```bash
npm install
```

## Running the Development Server

Start the development server by running:

```bash
npm run dev
```

Open your browser and visit the local URL displayed in the terminal (usually `http://localhost:5173`).

## Technologies Used

* React
* Vite
* JavaScript
* CSS
